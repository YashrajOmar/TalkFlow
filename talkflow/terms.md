## 1. Description of Service
TalkFlow is a local-first interview communication coach that runs in your browser or as a Chrome Extension. It records your speech, transcribes it using a local Whisper server (`faster-whisper`), and analyzes it using a local Ollama model (`llama3.2:3b`) — all on your own machine, with no required cloud connection.

Optionally, you may configure cloud providers (Google Gemini, OpenAI Whisper) for transcription or analysis. These are strictly optional and require your own API keys.
